# MomDuo Language Tutor Bot (@MomDuo_not)

基于 Telegram 和 Google Gemini AI 的德语、西班牙语和荷兰语学习助手。

## 🚀

支持发送图片（如书本、菜单、路牌），AI 将自动提取文字并分析。
* **使用方法**：
  * 通过 `Learn: [句子]` 触发句子解析。
  * 支持发送图片（如书本、菜单、路牌），AI 将自动提取文字并分析。
* **输出**：
    * 🏷️ **逐词直译**：贴近母语语序的翻译。
    * 📚 **名词**：冠词和复数形式。
    * 🔄 **动词**：常用时态的变位表。
    * 💡 **例句**：基于原句主语生成 3 个不同人称时态的例句。


## 🛠️ 

* **模型**: Google Gemini 2.5 Flash 
* **Server**: Render Background Worker
